使用物品帮助
输入 使用 (物品名称) 可以学习功法、神通,使用丹药,装备法器(注意功法等级，切莫低级替换高级) 输入 一键使用 (物品名称) 可以一键使用所有该物品 输入 换装 (物品名称) | 卸下 (物品名称) 可以卸下功法、神通、法器等
查询物品帮助
如果想要查看游戏内所有物品,输入 查看修仙界物品 [功法|神通|丹药|合成丹药|法器|防具] 即可查看(机器人回复可能稍慢,请耐心等待) 如果想要查看单个物品的详情信息,输入 查询 [物品名称] | 物品详情 [物品名称] 即可查看
炼金帮助
如果有多余的物品可以输入 炼金 (物品名称) 炼金可以获取灵石,也可以输入炼金 (物品名称) (炼金数量)来指定炼金多少个 你也可输入 一键炼金 【技能|装备|丹药|药材】 来一键炼金所有同类物品 如果你想炼金特定品级的药材,可以发送 药材一键炼金 后再发送品级

🏪 交易系统

坊市帮助
坊市分为【功法|辅修功法|神通|丹药|法器|防具|道具|药材】8个坊市 当你的灵石充足时,输入 查看坊市 (物品种类) (页码)可查看当前正在售卖的物品编号 如果找到心仪的物品,输入 坊市购买 (物品种类) (物品编号) (数量) 即可购买到背包 如果你想出售你的物品,输入 坊市上架 (物品名称) (金额) 即可上架到对应种类的坊市 如果你想下架你已经上架的物品,输入 坊市下架 (物品种类) (坊市物品编号) 即可下架对应物品
拍卖会帮助
每4小时系统会自动生成一场拍卖会 输入 拍卖品查看 | 查看拍卖品 可以查看等待拍卖的物品队列 输入 拍卖 (灵石数量) 即可竞价,30秒后价高者得 如果你想拍卖自己的物品输入 提交拍卖品 (物品名称) (起拍价) | 拍卖品提交 (物品名称) (起拍价) 即可在下一次拍卖会上优先拍卖你的物品 如果你想撤回你提交的拍卖品输入 撤回拍卖品 (拍卖品编号) 即可撤回你的拍卖品
灵庄帮助
灵庄是储存灵石的好地方,存在灵庄的灵石不会被其它道友抢或偷,还有一定的利率 输入 灵庄存灵石 (存入金额) 即可存入灵石 初始灵庄只能存入100w灵石,想要存入更多则输入 灵庄升级会员 消耗一定数量的灵石来提升会员等级 提升会员等级后可以提高灵庄可存入灵石数量以及利率,详情请输入 灵庄利息 | 利息 | 利息帮助 来查看 当你需要取出灵石时只需输入 灵庄取灵石 (取出金额)即可 如果你仅仅需要结算灵庄产生的利息则输入 灵庄结算 即可 注意: 存在灵庄的灵石是不可用于任何地方的！

🧘 修炼系统

修炼帮助
手动修炼:输入 修炼 或者 修仙 即可进行修炼,每两分钟可以修炼一次 灵石修炼:输入 灵石修炼 (灵石数量) 即可进行灵石修炼,每10个灵石可修炼1点修为 闭关修炼:输入 闭关 即可进入闭关修炼状态,输入 出关 即可结束闭关,闭关时间越长提升修为越高 灵石出关:输入 灵石出关 (灵石数量) 可稍微提高闭关获得的修为,灵石越多提升越高 双修:输入 双修@xxx (直接@qq就行)即可与道侣双修,双修只能与修为与自己相近或比自己低的道侣进行 输入 切磋 @xx 可以与道友切磋,不消耗气血 输入 我的双修次数 可查看剩余双修次数 注意:以上方法均可回复气血和真元
突破帮助
在一个境界修炼到一定修为则无法继续提升修为,此时需要突破 输入 突破 可查看当前突破信息 输入 直接突破 时突破失败会扣除一定修为 当背包有渡厄丹时可以输入 渡厄突破 突破失败不会扣除修为 当背包有渡厄金丹时可以输入 渡厄金丹突破 | 金丹突破 突破必定成功 全部境界请输入 境界帮助 | 境界 查看
